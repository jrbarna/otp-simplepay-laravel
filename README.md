# SimplePay for Laravel

## Installation

```bash
composer require jrbarna/simplepay-laravel
```
Publish config file

```php artisan vendor:publish --provider="Jrbarna\SimplePayLaravel\SimplePayLaravelServiceProvider" --tag="config"```
